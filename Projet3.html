À propos du projet

Trello est une application web de gestion de projets basée sur une interface Kanban qui permet aux utilisateurs d’organiser des tâches à travers des tableaux, des listes et des cartes. Elle permet de suivre les progrès, d’assigner des tâches, de définir des échéances et de collaborer avec une équipe en temps réel. L’application a une interface simple et intuitive, idéale pour des équipes de toutes tailles, depuis les petites startups jusqu’aux grandes entreprises.

  • Langages : HTML, CSS, JavaScript, TypeScript
  • Frameworks : React.js, Node.js, Express.js
  • Base de données : MongoDB
  • Outils de gestion de versions : Git, GitHub
